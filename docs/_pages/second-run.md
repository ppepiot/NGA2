Goal: make one intentional change and confirm the outcome changes.

## Suggested changes

Pick one of:

- Change a physical parameter (e.g., Reynolds number / viscosity)
- Change resolution
- Change a boundary condition

## Compare

Document what you compare (plots, norms, logs) and what “expected change” looks like.
